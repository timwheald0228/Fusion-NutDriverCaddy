# Fusion Nut Driver Caddy

A parametric Autodesk Fusion 360 script for creating custom tool organizers.

## Overview

This project generates a customizable nut driver organizer using the Fusion 360 API. The design is intended for 3D printing and can be modified through script parameters.

## Current Features

- Parametric base plate
- Rounded outside corners
- Configurable dimensions
- Fusion 360 Python script
- Designed for Klein hollow-shank nut drivers

## Planned Features

- Ergonomic carrying handle
- Flexible snap-in retaining clips
- Automatic driver spacing
- Engraved size labels
- Multiple layout options
- Support for additional tool types
